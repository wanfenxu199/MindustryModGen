📁 模组文件结构

生成模组后，你会看到这些文件和文件夹：

文件/文件夹 作用
mod.hjson 模组配置文件（名称、作者、版本等）
icon.png 模组图标（游戏里显示）
content/ 存放所有游戏逻辑定义（JSON 文件）
content/blocks/ 炮塔、方块、工厂等 JSON 文件
content/items/ 物品、材料等 JSON 文件
content/units/ 单位 JSON 文件
content/liquids/ 液体 JSON 文件
content/tech/ 科技树 JSON 文件
sprites/ 存放所有图片素材
sprites/blocks/ 炮塔/方块的贴图（PNG 文件）
sprites/items/ 物品的贴图（PNG 文件）
sprites/units/ 单位的贴图（PNG 文件）
sprites/liquids/ 液体的贴图（PNG 文件）
💡 提示：JSON 文件和 PNG 图片的名字要一致！
例如：super-turret.json → super-turret.png

🚀 快速开始

1. 填写模组基本信息（名称、作者、版本等）

2. 选择内容类型（炮塔/物品/方块/单位等）

3. 添加自定义字段

4. 点击「生成模组文件」

5. 可选：点击「打包 ZIP」导入游戏

💡 提示：字段名和值请参考下方各类型的参数说明

🎯 炮塔 (Turret)

ItemTurret - 物品炮塔

type: "ItemTurret"
name: "炮塔名称"
health: 800
size: 2 // 占地面积 1-5
range: 20 // 射程
reload: 10 // 装填时间（越小越快）
shootCone: 30 // 射击角度
shots: 1 // 每次射击弹丸数
shootWarmup: 0 // 预热时间
ammoType: "弹药类型"
ammoPerShot: 1
buildCost: {"copper": 100, "lead": 50} // 建造费用

DoubleTurret - 双管炮塔

type: "DoubleTurret"
name: "双管炮塔"
health: 1000
size: 3
range: 22
reload: 8
shots: 2 // 双管

LaserTurret - 激光炮塔

type: "LaserTurret"
name: "激光炮塔"
health: 600
size: 2
range: 15
reload: 20
damage: 10 // 每秒伤害

📋 炮塔常用参数

字段类型说明
health数字血量值
size数字占地面积 (1-5)
range数字攻击范围
reload数字装填时间 (越小越快)
shots数字每次射击弹丸数
shootCone数字射击扩散角度
shootWarmup数字预热时间
ammoPerShot数字每次消耗弹药数
buildCost对象建造费用

📦 物品 (Item)

type: "Item"
name: "物品名称"
flammability: 0.5 // 可燃性 0-1
explosiveness: 0.8 // 爆炸性 0-1
radioactivity: 0 // 放射性 0-1
hardness: 3 // 硬度 0-10
cost: 1 // 价值
color: "FF8800" // 颜色 (16进制)

📋 物品参数表

字段类型说明
flammability数字可燃性 0-1
explosiveness数字爆炸性 0-1
radioactivity数字放射性 0-1
hardness数字硬度 0-10
color字符串颜色 #RRGGBB

🏗️ 方块 (Block)

type: "Block"
name: "方块名称"
health: 400
size: 2
buildCost: {"copper": 50}
solid: true // 是否实体
buildTime: 60 // 建造时间

🔌 电力方块 (PowerBlock)

type: "PowerBlock"
name: "发电机"
powerProduction: 60 // 发电量
powerCapacity: 10000 // 储电量

🔫 单位 (Unit)

type: "Unit"
name: "单位名称"
health: 200
speed: 3 // 速度
damage: 15 // 攻击力
range: 10 // 攻击范围
buildCost: {"copper": 100}

💧 液体 (Liquid)

type: "Liquid"
name: "液体名称"
color: "4488FF"
temperature: 0.5 // 温度 0-1
viscosity: 0.3 // 粘稠度 0-1

💡 数据格式

✅ 数字：直接写 800

✅ 字符串：加引号 "名称"

✅ 布尔值：true / false

✅ 数组：["item1", "item2"]

✅ 对象：{"key": "value"}

❓ 常见问题

⚠️ 模组不显示？
检查 mod.hjson 中的 minGameVersion 是否 ≤ 你的游戏版本
⚠️ 炮塔不攻击？
检查是否有 range 和 ammoType 字段
